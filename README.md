## 🚀 Getting Started

### Prerequisites
- Python 3.9+
- Node.js 16+
- PostgreSQL

### Installation
```bash
# Clone repository
git clone https://github.com/Debprasad77/mind-metrics.git

# Backend setup
cd backend
pip install -r requirements.txt
python manage.py migrate

# Frontend setup
cd ../frontend
npm install
npm start
```
